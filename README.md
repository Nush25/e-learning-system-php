# e-learning-system-php
Web-based e-learning platform. It allows administrators to upload and manage lessons while students can access learning materials, download resources, and take quizzes online. Built using PHP, HTML, CSS, JavaScript, and MySQL.

## Project Overview
Web-based e-learning system developed for Vivekananda Prathishtan to manage learning resources digitally. The system helps automate the traditional learning process by allowing administrators to upload lessons and manage users, while students can access lessons and take quizzes online.

The main goal of this project is to improve accessibility to learning materials and make the education process more efficient through technology.

## Features

### Admin Module
- Upload lessons
- Edit lessons
- Delete lessons
- View lessons
- Manage users
- Add/Edit/Delete users

### Student Module
- Login and Logout
- View learning materials
- Download lessons
- Take quizzes

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL

## System Requirements

### Hardware
- Desktop or Laptop
- Internet connection (1 Mbps or higher recommended)

### Software
- Web Browser
- Code Editor (VS Code or similar)
- MySQL Database
- Local server (XAMPP/WAMP)

## Development Methodology
The project follows the **Agile Model**, which includes:

## Security
The system includes a secure login mechanism to protect user accounts and ensure safe access to learning resources.

## Objectives
- Provide access to learning resources anytime
- Remove traditional learning barriers

- Integrate technology into education

## Conclusion
Provides a simple and effective digital learning environment for educational institutions. It helps manage learning resources efficiently and improves accessibility for students and teachers.
